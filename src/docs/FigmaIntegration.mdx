# Figma Integration Guide

Connect components to [Figma Library](https://www.figma.com/design/S8zFdcOjt4qN4PrwntuCdt/Fanvue-Library) via Storybook Connect plugin.

## Quick Start

### Link Components to Storybook

1. **Get Storybook URL:**
   - CI automatically publishes Storybook on every PR/push to main
   - Find the Chromatic URL in PR comments or from the [Chromatic dashboard](https://www.chromatic.com/)

2. **Install Figma Plugin:**
   - In Figma: `Shift+I` â†’ Search for **"Storybook Connect"**
   - Or install from: https://www.figma.com/community/plugin/1056265616080331589

3. **Link Components:**
   - Select a component in Figma
   - Open Storybook Connect plugin (`Shift+I`)
   - Paste your Storybook URL
   - Find and link the matching story

> ðŸ’¡ **Tip:** Links persist automatically. Once linked, your team sees live Storybook stories in Figma Dev Mode.

## Design Tokens

Tokens live in `src/styles/styleTokens.json`, this file is generated via Style Dictionary.

**Workflow:**
1. Run `pnpm build:dictionary`
2. Add new tokens to `src/styles/theme.css`
